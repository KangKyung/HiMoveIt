### 구현 사항 
- 사용자가 시네마 그래프(사진, 혹은 영상)을 촬영
- 생성한 미디어 파일 확인 기능
- 시네마 그래프 콘텐츠 촬영 기능
- 촬영된 시네마 그래프에서 움직이는 영역을 선택 기능
- 생성된 시네마 그래프 콘텐츠를 볼 수 있는 기능
- 생성된 시네마 그래프 미디어 파일을 앱 외부로 공유(사진 앱 저장, 공유 기능 등)
